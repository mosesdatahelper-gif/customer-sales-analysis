# Customer Sales Analysis (SQL + Excel)

## Overview

This project analyzes customer purchasing behavior and product performance using SQL and Excel.

## Business Questions

* What is total revenue?
* Which customer generates the most revenue?
* Which product performs best?
* How does revenue change over time?

## Tools Used

* SQL (SQLite)
* Microsoft Excel

## Key Insights

* C1 is the highest-value customer
* Laptop is the top-performing product
* Revenue decreased from January to February
* Revenue is concentrated among top customers
